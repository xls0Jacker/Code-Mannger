# 2.17.1 
    箭头函数，基础知识：
        形如：
            let func = (arg1, arg2, ..., argN) => expression;
            <=>
            let func = function(arg1, arg2, ..., argN) {
                return expression;
            };
            （注意！如果我们只有一个参数，还可以省略掉参数外的圆括号；如果没有参数，括号则是空的（但括号必须保留））
        箭头函数可以像函数表达式一样使用。
# 2.17.2
    多行的箭头函数：
        加上花括号 {} 将内容括起来，其内按照原本函数的写法即可。
